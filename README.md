# 💻 Sistema de Gestão de Pedidos - Frontend

Frontend desenvolvido em **React + Vite + TypeScript + TailwindCSS**.

Aplicação responsável pela interface de:

- Login de usuários
- Cadastro de usuários
- Cadastro de clientes
- Cadastro de pedidos
- Finalização de pedidos
- Exclusão de pedidos com modal customizado

---

## 🚀 Tecnologias Utilizadas

- React
- TypeScript
- Vite
- React Router DOM
- Axios
- TailwindCSS
